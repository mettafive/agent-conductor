# Skill: api-doc-gen

**Shape:** iterates over each endpoint in endpoints.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **introspect** — introspect, parse route.
- **write doc** — write doc, document endpoint.
- **example requests** — example request, curl sample.  _(visibility divider — no hard gate needed)_
- **publish doc** — publish, openapi.

## Procedure
For each endpoint in endpoints:
  1. introspect — introspect, parse route.
  2. write doc — write doc, document endpoint.
  3. example requests — example request, curl sample.
  4. publish doc — publish, openapi.
