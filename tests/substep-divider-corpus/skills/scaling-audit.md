# Skill: scaling-audit

**Shape:** iterates over each service in services.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **collect metrics** — collect metrics, cpu memory.
- **analyze load** — analyze load, saturation.
- **cost estimate** — cost estimate, spend.  _(visibility divider — no hard gate needed)_
- **recommend size** — recommend, right-size.

## Procedure
For each service in services:
  1. collect metrics — collect metrics, cpu memory.
  2. analyze load — analyze load, saturation.
  3. cost estimate — cost estimate, spend.
  4. recommend size — recommend, right-size.
