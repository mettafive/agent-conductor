# Skill: lab-result-review

**Shape:** iterates over each panel in panels.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **ingest results** — ingest results, lab values.
- **flag abnormal** — flag abnormal, out of range.
- **trend compare** — trend, compare prior.  _(visibility divider — no hard gate needed)_
- **notify** — notify, alert clinician.

## Procedure
For each panel in panels:
  1. ingest results — ingest results, lab values.
  2. flag abnormal — flag abnormal, out of range.
  3. trend compare — trend, compare prior.
  4. notify — notify, alert clinician.
