# Skill: garden-plan

**Shape:** iterates over each bed in beds.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **assess soil** — assess soil, ph test.
- **select plants** — select plants, companion.
- **spacing layout** — spacing, layout.  _(visibility divider — no hard gate needed)_
- **schedule planting** — schedule planting, calendar.

## Procedure
For each bed in beds:
  1. assess soil — assess soil, ph test.
  2. select plants — select plants, companion.
  3. spacing layout — spacing, layout.
  4. schedule planting — schedule planting, calendar.
