# Skill: blog-pipeline

**Shape:** iterates over each topic in topics.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **outline** — outline, structure.
- **draft** — draft, write body.
- **seo titles** — seo title, meta description, slug.  _(visibility divider — no hard gate needed)_
- **internal links** — internal link, interlink, related posts.  _(visibility divider — no hard gate needed)_
- **publish** — publish, ship, cms.

## Procedure
For each topic in topics:
  1. outline — outline, structure.
  2. draft — draft, write body.
  3. seo titles — seo title, meta description, slug.
  4. internal links — internal link, interlink, related posts.
  5. publish — publish, ship, cms.
