# Skill: daily-enrichment

**Shape:** iterates over each clinic in clinics.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **research clinic** — research, crawl, gather data.
- **keyword research** — dataforseo, keyword research, search volume.  _(visibility divider — no hard gate needed)_
- **write fields** — write, description, story.
- **seo polish** — seo polish, akut, hembesök, meta.  _(visibility divider — no hard gate needed)_
- **save clinic** — save, patch, supabase, update db.
- **open pr** — pull request, open pr, ship.

## Procedure
For each clinic in clinics:
  1. research clinic — research, crawl, gather data.
  2. keyword research — dataforseo, keyword research, search volume.
  3. write fields — write, description, story.
  4. seo polish — seo polish, akut, hembesök, meta.
  5. save clinic — save, patch, supabase, update db.
  6. open pr — pull request, open pr, ship.
