# Skill: sourdough-cycle

**Shape:** iterates over each loaf in loaves.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **levain build** — levain, starter feed.
- **autolyse mix** — autolyse, mix dough.
- **fold schedule** — fold, stretch and fold.  _(visibility divider — no hard gate needed)_
- **retard** — retard, cold proof.  _(visibility divider — no hard gate needed)_
- **bake** — bake, score and bake.

## Procedure
For each loaf in loaves:
  1. levain build — levain, starter feed.
  2. autolyse mix — autolyse, mix dough.
  3. fold schedule — fold, stretch and fold.
  4. retard — retard, cold proof.
  5. bake — bake, score and bake.
