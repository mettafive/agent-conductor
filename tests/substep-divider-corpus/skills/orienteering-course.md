# Skill: orienteering-course

**Shape:** iterates over each control in controls.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **place control** — place control, feature pick.
- **set difficulty** — set difficulty, leg length.
- **legality check** — legality, out of bounds.  _(visibility divider — no hard gate needed)_
- **print control desc** — control description, clue sheet.

## Procedure
For each control in controls:
  1. place control — place control, feature pick.
  2. set difficulty — set difficulty, leg length.
  3. legality check — legality, out of bounds.
  4. print control desc — control description, clue sheet.
