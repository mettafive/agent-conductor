# Skill: ad-compliance

**Shape:** iterates over each ad in ads.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **scan ad** — scan ad, read creative.
- **check claims** — check claims, substantiation.
- **disclosure check** — disclosure, fine print.  _(visibility divider — no hard gate needed)_
- **approve ad** — approve, verdict.

## Procedure
For each ad in ads:
  1. scan ad — scan ad, read creative.
  2. check claims — check claims, substantiation.
  3. disclosure check — disclosure, fine print.
  4. approve ad — approve, verdict.
