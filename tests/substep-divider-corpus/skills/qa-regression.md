# Skill: qa-regression

**Shape:** iterates over each flow in flows.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **setup fixture** — setup fixture, test data.
- **run flow** — run flow, execute test.
- **screenshot diff** — screenshot diff, visual regression.  _(visibility divider — no hard gate needed)_
- **report result** — report, pass fail.

## Procedure
For each flow in flows:
  1. setup fixture — setup fixture, test data.
  2. run flow — run flow, execute test.
  3. screenshot diff — screenshot diff, visual regression.
  4. report result — report, pass fail.
