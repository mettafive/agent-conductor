# Skill: accessibility-sweep

**Shape:** iterates over each view in views.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **render view** — render, load view.
- **audit wcag** — audit, wcag.
- **contrast check** — contrast, color ratio.  _(visibility divider — no hard gate needed)_
- **aria check** — aria, screen reader.  _(visibility divider — no hard gate needed)_
- **file issues** — file issues, log violations.

## Procedure
For each view in views:
  1. render view — render, load view.
  2. audit wcag — audit, wcag.
  3. contrast check — contrast, color ratio.
  4. aria check — aria, screen reader.
  5. file issues — file issues, log violations.
