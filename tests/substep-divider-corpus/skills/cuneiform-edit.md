# Skill: cuneiform-edit

**Shape:** iterates over each tablet in tablets.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **transliterate** — transliterate, sign reading.
- **normalize akkadian** — normalize, grammatical.
- **mark breaks** — mark breaks, lacuna.  _(visibility divider — no hard gate needed)_
- **sign list ref** — sign list, borger.  _(visibility divider — no hard gate needed)_
- **publish edition** — publish edition, translation.

## Procedure
For each tablet in tablets:
  1. transliterate — transliterate, sign reading.
  2. normalize akkadian — normalize, grammatical.
  3. mark breaks — mark breaks, lacuna.
  4. sign list ref — sign list, borger.
  5. publish edition — publish edition, translation.
