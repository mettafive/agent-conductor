# Skill: resume-screen

**Shape:** iterates over each candidate in candidates.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **parse resume** — parse resume, extract.
- **match criteria** — match, rubric.
- **bias check** — bias check, fairness.  _(visibility divider — no hard gate needed)_
- **rank** — rank, shortlist.

## Procedure
For each candidate in candidates:
  1. parse resume — parse resume, extract.
  2. match criteria — match, rubric.
  3. bias check — bias check, fairness.
  4. rank — rank, shortlist.
