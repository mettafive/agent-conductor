# Skill: churn-outreach

**Shape:** iterates over each account in accounts.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **score risk** — score risk, churn score.
- **draft outreach** — draft outreach, win-back email.
- **offer pick** — offer, incentive.  _(visibility divider — no hard gate needed)_
- **queue send** — queue, schedule send.

## Procedure
For each account in accounts:
  1. score risk — score risk, churn score.
  2. draft outreach — draft outreach, win-back email.
  3. offer pick — offer, incentive.
  4. queue send — queue, schedule send.
