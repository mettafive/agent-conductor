# Skill: menu-translation

**Shape:** iterates over each item in items.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **translate name** — translate name, dish name.
- **describe** — describe, description.
- **allergen note** — allergen, dietary.  _(visibility divider — no hard gate needed)_
- **save menu item** — save, store item.

## Procedure
For each item in items:
  1. translate name — translate name, dish name.
  2. describe — describe, description.
  3. allergen note — allergen, dietary.
  4. save menu item — save, store item.
