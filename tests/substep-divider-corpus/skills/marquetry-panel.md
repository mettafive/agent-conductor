# Skill: marquetry-panel

**Shape:** iterates over each panel in panels.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **cut veneer** — cut veneer, fret saw.
- **assemble pattern** — assemble, fit pieces.
- **sand shade** — sand shade, hot sand.  _(visibility divider — no hard gate needed)_
- **glue press** — glue, press panel.

## Procedure
For each panel in panels:
  1. cut veneer — cut veneer, fret saw.
  2. assemble pattern — assemble, fit pieces.
  3. sand shade — sand shade, hot sand.
  4. glue press — glue, press panel.
