# Skill: perfume-accord

**Shape:** iterates over each accord in accords.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **select notes** — select notes, ingredients.
- **balance pyramid** — balance, top heart base.
- **ifra check** — ifra, regulatory limit.  _(visibility divider — no hard gate needed)_
- **record formula** — record formula, percentages.

## Procedure
For each accord in accords:
  1. select notes — select notes, ingredients.
  2. balance pyramid — balance, top heart base.
  3. ifra check — ifra, regulatory limit.
  4. record formula — record formula, percentages.
