# Skill: glassblowing-vessel

**Shape:** iterates over each vessel in vessels.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **gather gob** — gather, gob.
- **shape form** — shape, marver block.
- **color apply** — color, frit roll.  _(visibility divider — no hard gate needed)_
- **anneal** — anneal, lehr.

## Procedure
For each vessel in vessels:
  1. gather gob — gather, gob.
  2. shape form — shape, marver block.
  3. color apply — color, frit roll.
  4. anneal — anneal, lehr.
