# Skill: scrimshaw-engrave

**Shape:** iterates over each piece in pieces.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **design transfer** — design transfer, trace pattern.
- **engrave** — engrave, scribe lines.
- **ink rub** — ink, pigment rub.  _(visibility divider — no hard gate needed)_
- **seal piece** — seal, finish.

## Procedure
For each piece in pieces:
  1. design transfer — design transfer, trace pattern.
  2. engrave — engrave, scribe lines.
  3. ink rub — ink, pigment rub.
  4. seal piece — seal, finish.
