# Skill: release-notes

**Shape:** single pass.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **collect prs** — collect prs, merged commits, git log.
- **categorize** — categorize, group by type.
- **write notes** — write notes, draft changelog.
- **highlight breaking** — breaking change, migration note.  _(visibility divider — no hard gate needed)_
- **publish notes** — publish, tag release.

## Procedure
1. collect prs — collect prs, merged commits, git log.
2. categorize — categorize, group by type.
3. write notes — write notes, draft changelog.
4. highlight breaking — breaking change, migration note.
5. publish notes — publish, tag release.
