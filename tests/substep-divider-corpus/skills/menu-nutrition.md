# Skill: menu-nutrition

**Shape:** iterates over each item in items.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **lookup ingredients** — lookup ingredients, components.
- **compute nutrition** — compute nutrition, calories macros.
- **allergen label** — allergen, label.  _(visibility divider — no hard gate needed)_
- **publish nutrition** — publish, save label.

## Procedure
For each item in items:
  1. lookup ingredients — lookup ingredients, components.
  2. compute nutrition — compute nutrition, calories macros.
  3. allergen label — allergen, label.
  4. publish nutrition — publish, save label.
