# Skill: menu-engineering

**Shape:** iterates over each dish in dishes.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **cost recipe** — cost recipe, ingredient cost.
- **price dish** — price dish, set price.
- **margin flag** — margin, profitability.  _(visibility divider — no hard gate needed)_
- **update menu** — update menu, publish.

## Procedure
For each dish in dishes:
  1. cost recipe — cost recipe, ingredient cost.
  2. price dish — price dish, set price.
  3. margin flag — margin, profitability.
  4. update menu — update menu, publish.
