# Skill: product-copy

**Shape:** iterates over each product in products.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **gather specs** — gather specs, spec sheet.
- **write copy** — write copy, marketing copy.
- **tone pass** — tone, brand voice.  _(visibility divider — no hard gate needed)_
- **localize** — localize, translation, locale.  _(visibility divider — no hard gate needed)_
- **upload** — upload, save to catalog.

## Procedure
For each product in products:
  1. gather specs — gather specs, spec sheet.
  2. write copy — write copy, marketing copy.
  3. tone pass — tone, brand voice.
  4. localize — localize, translation, locale.
  5. upload — upload, save to catalog.
