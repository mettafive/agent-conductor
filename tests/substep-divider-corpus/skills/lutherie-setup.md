# Skill: lutherie-setup

**Shape:** iterates over each instrument in instruments.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **measure action** — measure action, string height.
- **adjust truss** — adjust truss, neck relief.
- **intonation** — intonation, saddle.  _(visibility divider — no hard gate needed)_
- **nut slots** — nut slot, file nut.  _(visibility divider — no hard gate needed)_
- **final tune** — final tune, string up.

## Procedure
For each instrument in instruments:
  1. measure action — measure action, string height.
  2. adjust truss — adjust truss, neck relief.
  3. intonation — intonation, saddle.
  4. nut slots — nut slot, file nut.
  5. final tune — final tune, string up.
