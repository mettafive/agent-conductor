# Skill: ticket-dedup

**Shape:** iterates over each cluster in clusters.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **cluster** — cluster, group tickets.
- **merge** — merge, dedupe tickets.
- **link context** — link context, preserve links.  _(visibility divider — no hard gate needed)_
- **close dupes** — close, resolve duplicates.

## Procedure
For each cluster in clusters:
  1. cluster — cluster, group tickets.
  2. merge — merge, dedupe tickets.
  3. link context — link context, preserve links.
  4. close dupes — close, resolve duplicates.
