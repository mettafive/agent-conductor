**Shape:** iterates over each module in modules.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **build module** — build module, content.
- **add quiz** — add quiz, knowledge check.
- **track completion** — track completion, attestation.  _(visibility divider — no hard gate needed)_
- **publish module** — publish, assign.
