# Skill: course-build

**Shape:** iterates over each lesson in lessons.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **outline lesson** — outline lesson, learning objectives.
- **write content** — write content, lesson body.
- **quiz** — quiz, assessment.  _(visibility divider — no hard gate needed)_
- **transcript** — transcript, captions.  _(visibility divider — no hard gate needed)_
- **publish lesson** — publish, lms.

## Procedure
For each lesson in lessons:
  1. outline lesson — outline lesson, learning objectives.
  2. write content — write content, lesson body.
  3. quiz — quiz, assessment.
  4. transcript — transcript, captions.
  5. publish lesson — publish, lms.
