# Skill: data-quality

**Shape:** iterates over each dataset in datasets.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **profile data** — profile, describe dataset.
- **run checks** — run checks, quality rules.
- **anomaly flag** — anomaly, outlier.  _(visibility divider — no hard gate needed)_
- **report quality** — report, scorecard.

## Procedure
For each dataset in datasets:
  1. profile data — profile, describe dataset.
  2. run checks — run checks, quality rules.
  3. anomaly flag — anomaly, outlier.
  4. report quality — report, scorecard.
