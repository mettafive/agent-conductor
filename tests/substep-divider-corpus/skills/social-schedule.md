# Skill: social-schedule

**Shape:** iterates over each post in posts.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **draft post** — draft post, compose.
- **attach media** — attach media, image video.
- **hashtag** — hashtag, tags.  _(visibility divider — no hard gate needed)_
- **best time** — best time, optimal slot.  _(visibility divider — no hard gate needed)_
- **schedule post** — schedule, queue post.

## Procedure
For each post in posts:
  1. draft post — draft post, compose.
  2. attach media — attach media, image video.
  3. hashtag — hashtag, tags.
  4. best time — best time, optimal slot.
  5. schedule post — schedule, queue post.
