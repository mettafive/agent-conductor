# Skill: image-pipeline

**Shape:** iterates over each article in articles.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **prompt build** — build prompt, image prompt.
- **generate** — generate, render image.
- **alt text** — alt text, accessibility caption.  _(visibility divider — no hard gate needed)_
- **compress** — compress, optimize size.  _(visibility divider — no hard gate needed)_
- **upload cdn** — upload, cdn, storage.

## Procedure
For each article in articles:
  1. prompt build — build prompt, image prompt.
  2. generate — generate, render image.
  3. alt text — alt text, accessibility caption.
  4. compress — compress, optimize size.
  5. upload cdn — upload, cdn, storage.
