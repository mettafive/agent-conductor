# Skill: ikebana-arrange

**Shape:** iterates over each arrangement in arrangements.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **select material** — select material, branches flowers.
- **set structure** — set structure, shin soe hikae.
- **kenzan fix** — kenzan, pin holder.  _(visibility divider — no hard gate needed)_
- **finalize arrangement** — finalize, display.

## Procedure
For each arrangement in arrangements:
  1. select material — select material, branches flowers.
  2. set structure — set structure, shin soe hikae.
  3. kenzan fix — kenzan, pin holder.
  4. finalize arrangement — finalize, display.
