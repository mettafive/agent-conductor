# Skill: vendor-onboard

**Shape:** iterates over each vendor in vendors.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **collect docs** — collect docs, w9 insurance.
- **verify vendor** — verify, background check.
- **risk tier** — risk tier, classify risk.  _(visibility divider — no hard gate needed)_
- **activate** — activate, approve vendor.

## Procedure
For each vendor in vendors:
  1. collect docs — collect docs, w9 insurance.
  2. verify vendor — verify, background check.
  3. risk tier — risk tier, classify risk.
  4. activate — activate, approve vendor.
