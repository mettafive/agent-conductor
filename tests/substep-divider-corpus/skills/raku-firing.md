# Skill: raku-firing

**Shape:** iterates over each piece in pieces.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **bisque load** — bisque, load kiln.
- **glaze** — glaze, apply glaze.
- **reduction plan** — reduction, post-fire.  _(visibility divider — no hard gate needed)_
- **fire quench** — fire, quench.

## Procedure
For each piece in pieces:
  1. bisque load — bisque, load kiln.
  2. glaze — glaze, apply glaze.
  3. reduction plan — reduction, post-fire.
  4. fire quench — fire, quench.
