# Skill: kb-refresh

**Shape:** iterates over each article in articles.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **audit article** — audit, check staleness.
- **update** — update, rewrite.
- **screenshot refresh** — screenshot, refresh image.  _(visibility divider — no hard gate needed)_
- **republish** — republish, save.

## Procedure
For each article in articles:
  1. audit article — audit, check staleness.
  2. update — update, rewrite.
  3. screenshot refresh — screenshot, refresh image.
  4. republish — republish, save.
