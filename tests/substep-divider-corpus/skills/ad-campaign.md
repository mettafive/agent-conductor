# Skill: ad-campaign

**Shape:** iterates over each audience in audiences.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **define audience** — define audience, targeting.
- **write creative** — write creative, ad copy.
- **budget split** — budget, spend allocation.  _(visibility divider — no hard gate needed)_
- **launch** — launch, publish campaign.

## Procedure
For each audience in audiences:
  1. define audience — define audience, targeting.
  2. write creative — write creative, ad copy.
  3. budget split — budget, spend allocation.
  4. launch — launch, publish campaign.
