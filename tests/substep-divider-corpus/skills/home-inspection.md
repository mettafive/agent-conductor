# Skill: home-inspection

**Shape:** iterates over each room in rooms.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **inspect room** — inspect, walkthrough.
- **document issues** — document issues, defects.
- **photo evidence** — photo evidence, capture.  _(visibility divider — no hard gate needed)_
- **compile report** — compile report, summary.

## Procedure
For each room in rooms:
  1. inspect room — inspect, walkthrough.
  2. document issues — document issues, defects.
  3. photo evidence — photo evidence, capture.
  4. compile report — compile report, summary.
