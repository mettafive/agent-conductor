# Skill: knot-tying-guide

**Shape:** iterates over each knot in knots.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **describe use** — describe use, purpose.
- **write steps** — write steps, tying sequence.
- **strength note** — strength, breaking load.  _(visibility divider — no hard gate needed)_
- **add diagram** — add diagram, illustration.

## Procedure
For each knot in knots:
  1. describe use — describe use, purpose.
  2. write steps — write steps, tying sequence.
  3. strength note — strength, breaking load.
  4. add diagram — add diagram, illustration.
