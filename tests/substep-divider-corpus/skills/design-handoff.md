# Skill: design-handoff

**Shape:** iterates over each screen in screens.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **export assets** — export assets, slice.
- **spec redlines** — redline, spacing spec.
- **token map** — design token, variable map.  _(visibility divider — no hard gate needed)_
- **publish handoff** — publish handoff, share.

## Procedure
For each screen in screens:
  1. export assets — export assets, slice.
  2. spec redlines — redline, spacing spec.
  3. token map — design token, variable map.
  4. publish handoff — publish handoff, share.
