# Skill: loan-underwrite

**Shape:** iterates over each application in applications.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **pull credit** — pull credit, credit report.
- **assess risk** — assess risk, underwrite.
- **dti compute** — debt to income, dti ratio.  _(visibility divider — no hard gate needed)_
- **decision** — decision, approve decline.

## Procedure
For each application in applications:
  1. pull credit — pull credit, credit report.
  2. assess risk — assess risk, underwrite.
  3. dti compute — debt to income, dti ratio.
  4. decision — decision, approve decline.
