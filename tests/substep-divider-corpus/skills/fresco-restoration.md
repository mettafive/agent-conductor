# Skill: fresco-restoration

**Shape:** iterates over each panel in panels.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **survey damage** — survey damage, condition map.
- **consolidate** — consolidate, stabilize plaster.
- **retouch tratteggio** — tratteggio, in-painting.  _(visibility divider — no hard gate needed)_
- **uv document** — uv, document layers.  _(visibility divider — no hard gate needed)_
- **final varnish** — varnish, protective coat.

## Procedure
For each panel in panels:
  1. survey damage — survey damage, condition map.
  2. consolidate — consolidate, stabilize plaster.
  3. retouch tratteggio — tratteggio, in-painting.
  4. uv document — uv, document layers.
  5. final varnish — varnish, protective coat.
