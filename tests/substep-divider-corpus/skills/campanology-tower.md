# Skill: campanology-tower

**Shape:** iterates over each bell in bells.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **inspect bell** — inspect, headstock.
- **tune bell** — tune, shave harmonic.
- **clapper set** — clapper, flight adjust.  _(visibility divider — no hard gate needed)_
- **rehang** — rehang, install.

## Procedure
For each bell in bells:
  1. inspect bell — inspect, headstock.
  2. tune bell — tune, shave harmonic.
  3. clapper set — clapper, flight adjust.
  4. rehang — rehang, install.
