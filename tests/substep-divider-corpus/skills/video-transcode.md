# Skill: video-transcode

**Shape:** iterates over each clip in clips.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **probe** — probe, inspect clip.
- **transcode** — transcode, encode.
- **thumbnail** — thumbnail, poster frame.  _(visibility divider — no hard gate needed)_
- **subtitle** — subtitle, caption track.  _(visibility divider — no hard gate needed)_
- **store asset** — store, save asset.

## Procedure
For each clip in clips:
  1. probe — probe, inspect clip.
  2. transcode — transcode, encode.
  3. thumbnail — thumbnail, poster frame.
  4. subtitle — subtitle, caption track.
  5. store asset — store, save asset.
