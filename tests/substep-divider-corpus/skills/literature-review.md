# Skill: literature-review

**Shape:** iterates over each paper in papers.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **fetch paper** — fetch, download paper.
- **summarize** — summarize, extract findings.
- **cite extract** — citation, bibtex, reference.  _(visibility divider — no hard gate needed)_
- **synthesize** — synthesize, combine findings.

## Procedure
For each paper in papers:
  1. fetch paper — fetch, download paper.
  2. summarize — summarize, extract findings.
  3. cite extract — citation, bibtex, reference.
  4. synthesize — synthesize, combine findings.
