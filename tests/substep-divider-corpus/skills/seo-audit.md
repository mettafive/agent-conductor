# Skill: seo-audit

**Shape:** iterates over each page in pages.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **crawl page** — crawl, fetch page.
- **audit meta** — audit meta, title description.
- **schema check** — schema, json-ld, structured data.  _(visibility divider — no hard gate needed)_
- **speed check** — speed, core web vitals, lcp.  _(visibility divider — no hard gate needed)_
- **recommend fixes** — recommend, fixes.

## Procedure
For each page in pages:
  1. crawl page — crawl, fetch page.
  2. audit meta — audit meta, title description.
  3. schema check — schema, json-ld, structured data.
  4. speed check — speed, core web vitals, lcp.
  5. recommend fixes — recommend, fixes.
