# Skill: falconry-weight

**Shape:** iterates over each bird in birds.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **weigh bird** — weigh, flying weight.
- **assess condition** — assess condition, mews check.
- **ration plan** — ration, casting.  _(visibility divider — no hard gate needed)_
- **log bird** — log, weight record.

## Procedure
For each bird in birds:
  1. weigh bird — weigh, flying weight.
  2. assess condition — assess condition, mews check.
  3. ration plan — ration, casting.
  4. log bird — log, weight record.
