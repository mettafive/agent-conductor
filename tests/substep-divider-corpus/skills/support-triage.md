# Skill: support-triage

**Shape:** iterates over each ticket in tickets.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **read ticket** — read ticket, load ticket.
- **classify** — classify, categorize.
- **sentiment** — sentiment, urgency.  _(visibility divider — no hard gate needed)_
- **route** — route, assign queue.
- **draft reply** — draft reply, respond.

## Procedure
For each ticket in tickets:
  1. read ticket — read ticket, load ticket.
  2. classify — classify, categorize.
  3. sentiment — sentiment, urgency.
  4. route — route, assign queue.
  5. draft reply — draft reply, respond.
