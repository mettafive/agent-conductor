# Skill: podcast-publish

**Shape:** iterates over each episode in episodes.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **transcribe** — transcribe, transcript.
- **show notes** — show notes, episode notes.
- **chapter marks** — chapter, timestamps.  _(visibility divider — no hard gate needed)_
- **publish feed** — publish, rss feed.

## Procedure
For each episode in episodes:
  1. transcribe — transcribe, transcript.
  2. show notes — show notes, episode notes.
  3. chapter marks — chapter, timestamps.
  4. publish feed — publish, rss feed.
