# Skill: email-campaign-qa

**Shape:** iterates over each email in emails.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **render email** — render, preview email.
- **link check** — link check, verify links.
- **spam score** — spam score, deliverability.  _(visibility divider — no hard gate needed)_
- **approve** — approve, sign off.

## Procedure
For each email in emails:
  1. render email — render, preview email.
  2. link check — link check, verify links.
  3. spam score — spam score, deliverability.
  4. approve — approve, sign off.
