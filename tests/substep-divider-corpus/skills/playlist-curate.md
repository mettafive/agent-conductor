# Skill: playlist-curate

**Shape:** iterates over each mood in moods.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **seed tracks** — seed tracks, starting songs.
- **expand** — expand, recommend tracks.
- **flow order** — flow order, sequence, transitions.  _(visibility divider — no hard gate needed)_
- **save playlist** — save playlist, publish.

## Procedure
For each mood in moods:
  1. seed tracks — seed tracks, starting songs.
  2. expand — expand, recommend tracks.
  3. flow order — flow order, sequence, transitions.
  4. save playlist — save playlist, publish.
