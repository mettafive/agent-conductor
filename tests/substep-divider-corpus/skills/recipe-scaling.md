# Skill: recipe-scaling

**Shape:** iterates over each recipe in recipes.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **read recipe** — read recipe, ingredients.
- **scale** — scale, adjust quantities.
- **unit convert** — unit convert, metric imperial.  _(visibility divider — no hard gate needed)_
- **save recipe** — save, store recipe.

## Procedure
For each recipe in recipes:
  1. read recipe — read recipe, ingredients.
  2. scale — scale, adjust quantities.
  3. unit convert — unit convert, metric imperial.
  4. save recipe — save, store recipe.
