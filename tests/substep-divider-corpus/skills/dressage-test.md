# Skill: dressage-test

**Shape:** iterates over each movement in movements.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **read movement** — read movement, test sheet.
- **score movement** — score, mark.
- **collective marks** — collective marks, impulsion.  _(visibility divider — no hard gate needed)_
- **tally** — tally, final percentage.

## Procedure
For each movement in movements:
  1. read movement — read movement, test sheet.
  2. score movement — score, mark.
  3. collective marks — collective marks, impulsion.
  4. tally — tally, final percentage.
