# Skill: menu-photo-shoot

**Shape:** iterates over each dish in dishes.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **plate** — plate, styling.
- **shoot** — shoot, capture photo.
- **retouch** — retouch, color grade.  _(visibility divider — no hard gate needed)_
- **upload photo** — upload, save photo.

## Procedure
For each dish in dishes:
  1. plate — plate, styling.
  2. shoot — shoot, capture photo.
  3. retouch — retouch, color grade.
