# Skill: contract-review

**Shape:** iterates over each contract in contracts.

**Distinct work-units this skill performs** (each is a phase the user will look for on the board):

- **extract clauses** — extract clauses, parse terms.
- **flag risks** — flag risk, risky clause.
- **redline suggest** — redline, suggested edit.  _(visibility divider — no hard gate needed)_
- **summarize contract** — summarize, brief.

## Procedure
For each contract in contracts:
  1. extract clauses — extract clauses, parse terms.
  2. flag risks — flag risk, risky clause.
  3. redline suggest — redline, suggested edit.
  4. summarize contract — summarize, brief.
